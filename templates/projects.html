Sanicを使用したWebポートフォリオサイト

開発期間: 2025年4月〜5月 使用技術: Python, Sanic, HTML, CSS, JavaScript

2000年代のインターネット文化を再現したレトロなポートフォリオサイト（このサイト）です。

Sanicフレームワークを使用して構築し、Jinja2テンプレートエンジンでHTMLレンダリングを行っています。

訪問者カウンター、時計表示、マーキータグなど、懐かしいWeb要素を再現しました。

主な機能：

  • レトロなデザインとレイアウト
  • 訪問者カウンター
  • リアルタイム時計表示
  • 2000年代風のナビゲーション

オンラインメモ帳アプリケーション

開発期間: 2024年10月〜2025年1月 使用技術: Python, Flask, SQLite, JavaScript, Bootstrap

ブラウザ上で動作するシンプルなメモ帳アプリケーションです。

Flaskフレームワークを使用し、SQLiteデータベースでメモを管理します。

ユーザー認証機能を実装し、メモの保存、編集、削除、検索が可能です。

主な機能：

  • ユーザー登録・ログイン機能
  • メモの作成・編集・削除
  • タグ付け機能
  • 全文検索機能
  • メモの共有機能

天気予報API連携Webアプリ

開発期間: 2024年7月〜9月 使用技術: Python, Django, React, OpenWeatherMap API

OpenWeatherMap APIを利用した天気予報表示アプリケーションです。

DjangoとReactを組み合わせて構築し、世界中の都市の天気情報をリアルタイムで取得、表示します。

都市の追加、削除、お気に入り登録などの機能を実装しました。

主な機能：

  • 世界中の都市の天気情報表示
  • 5日間の天気予報表示
  • 都市のお気に入り登録
  • 気温、湿度、風速などの詳細情報表示
  • 地図上での天気表示
